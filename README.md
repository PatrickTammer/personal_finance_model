# 10-Year Financial Dashboard

Static financial model and dashboard for projecting salary, taxes, savings, life events, net assets, and U.S. wealth percentile.

## Open Locally

Open `interactive-dashboard/index.html` in a browser, or serve the folder with a small local web server.

## GitHub Pages

Publish this repo with GitHub Pages from the `main` branch root. The root `index.html` redirects to the dashboard app.
